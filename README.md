# Anjali Kamal

A modern and interactive portfolio website showcasing my work, skills, and creative journey.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

### Clone the repository

```sh
git clone https://github.com/A-verse/a-versee.git
```

### Navigate to the project directory

```sh
cd a-versee
```

### Install dependencies

```sh
npm install
```

### Start the development server

```sh
npm run dev
```

### Build for production

```sh
npm run build
```

### Preview production build

```sh
npm run preview
```

- Responsive design
- Smooth animations and interactions
- Modern UI
